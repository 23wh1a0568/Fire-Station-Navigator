# 🚒 Fire Rescue Navigator

## 📌 Overview
Fire Rescue Navigator is a grid-based simulation where an AI-driven fire truck navigates to rescue people from a burning house using the shortest path.

---

## 🚀 Features
- AI-based shortest path navigation (BFS)
- Grid-based environment
- Random obstacle placement
- Real-time fire truck movement
- Sound effects (siren)
- GUI using Tkinter

---

## 🛠️ Tech Stack
- Python
- Tkinter
- Pillow (PIL)
- Pygame
